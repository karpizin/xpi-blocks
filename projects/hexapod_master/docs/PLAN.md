# План реализации Hexapod Master

Проект разбит на 5 этапов, каждый из которых добавляет новый уровень интеллектуального поведения.

## Этап 1: Геометрия и IK одной лапы (The Leg)
*   [x] Математический расчет IK для 3-звенной лапы (Coxa, Femur, Tibia). См. `docs/LEG_IK_MATH.md`.
*   [x] Создание URDF модели одной лапы для визуализации. См. `urdf/leg.urdf`.
*   [x] Реализация интерактивного узла `leg_ik_viewer_node.py`, связывающего IK-класс с визуализацией в реальном времени.
*   [x] Тестирование в Rviz2: перемещение лапы по заданным (x, y, z) координатам.

## Этап 2: Кинематика корпуса (The Body)
*   [x] Реализация класса `BodyKinematics` для расчета трансформаций всех 6 лап. См. `scripts/body_kinematics.py`.
*   [x] Создание конфигурационного файла геометрии `config/hexapod_config.yaml`.
*   [x] Создание интеграционного узла `hexapod_body_kinematics_node.py`.
*   [x] Полная URDF модель гексапода (6 лап на корпусе). См. `urdf/hexapod.urdf`.
*   [x] Реализация плавного "приседания" и наклонов через топик `/hexapod/body_pose`.

## Этап 3: Физическая симуляция (The Void)
*   [ ] Настройка URDF для Gazebo (добавление масс, инерции и коллизий).
*   [ ] Конфигурация `ros2_control` для управления 18-ю суставами.
*   [ ] Создание Launch-файла для запуска Gazebo и "спавна" робота.
*   [ ] Тестирование устойчивости в симуляторе при различных позах корпуса.
