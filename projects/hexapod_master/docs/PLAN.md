# План реализации Hexapod Master

Проект разбит на 5 этапов, каждый из которых добавляет новый уровень интеллектуального поведения.

## Этап 1: Геометрия и IK одной лапы (The Leg)
*   [x] Математический расчет IK для 3-звенной лапы (Coxa, Femur, Tibia). См. `docs/LEG_IK_MATH.md`.
*   [x] Создание URDF модели одной лапы для визуализации. См. `urdf/leg.urdf`.
*   [ ] Реализация интерактивного узла `leg_ik_viewer_node.py`, связывающего IK-класс с визуализацией в реальном времени.
*   [ ] Тестирование в Rviz2: перемещение лапы по заданным (x, y, z) координатам.

## Этап 2: Кинематика корпуса (The Body)
*   [ ] Расчет матрицы трансформации для 6 лап относительно центра масс.
*   [ ] Реализация управления ориентацией корпуса (Pitch, Roll, Yaw) и высотой.
*   [ ] Интеграция IMU (MPU6050) для удержания горизонта.

## Этап 3: Генератор походки (The Gait)
*   [ ] Реализация классического Tripod Gait (шаг 3х3).
*   [ ] Реализация Ripple Gait для более устойчивого перемещения.
*   [ ] Плавная интерполяция траектории кончика лапы (Bezier curves).

## Этап 4: Машинное обучение и Плавность (The Soul)
*   [ ] Подготовка среды симуляции (PyBullet или Isaac Gym).
*   [ ] Обучение модели Reinforcement Learning для адаптации к неровному рельефу.
*   [ ] Дистилляция модели в легковесную сеть для запуска на Raspberry Pi.
*   [ ] Реализация "органического" движения (микродвижения корпуса при ходьбе).

## Этап 5: Манипуляция и VLM (The Action)
*   [ ] Интеграция IK манипулятора.
*   [ ] Координация центра масс при выносе манипулятора вперед (предотвращение опрокидывания).
*   [ ] Связка с VLM для захвата объектов по визуальным координатам.
