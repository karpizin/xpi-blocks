# Рекомендуемое оборудование для Meshtastic LoRa Mesh

Этот документ содержит обзор оборудования, совместимого с модулем `Meshtastic Swarm`, разделенный по типам использования: для бортовых систем роботов, наземных станций и интеграции с Raspberry Pi.

---

## 1. Автономные модули (Standalone)
Лучший выбор для интеграции через USB или создания мобильных узлов связи.

### 1.1 Heltec WiFi LoRa 32 (V3)
*   **Чип:** ESP32-S3 + Semtech SX1262.
*   **Плюсы:** Наличие 0.96" OLED экрана (удобно для диагностики на месте), низкая цена, высокая доступность.
*   **Использование:** Наземные станции, отладочные узлы.
*   **Интерфейс:** USB-C (Serial).

### 1.2 LilyGO T-Beam (V1.1 / V1.2)
*   **Чип:** ESP32 + SX1276/SX1262 + GPS (Neo-6M/M8N).
*   **Плюсы:** Интегрированный GPS и держатель для аккумулятора 18650.
*   **Использование:** Полноценный мобильный узел "все-в-одном".
*   **Минус:** Относительно тяжелый для установки на мелкие дроны.

### 1.3 RAK Wireless WisBlock (RAK4631)
*   **Чип:** nRF52840 (Bluetooth 5.0) + SX1262.
*   **Плюсы:** **Экстремально низкое энергопотребление**. Модульная конструкция (можно добавить датчики или другой интерфейс).
*   **Использование:** Идеален для бортовых систем дронов, где важен вес и время жизни от батареи.
*   **Интерфейс:** UART / USB.

---

## 2. Модули расширения (Hats) для Raspberry Pi
Если робот управляется Raspberry Pi, удобнее использовать Hats для прямого подключения к GPIO.

### 2.1 Waveshare SX1262 LoRa HAT
*   **Плюсы:** Прямое подключение к 40-pin GPIO. Хорошая документация и поддержка частот 868/915 МГц.
*   **Интерфейс:** SPI (в Meshtastic может потребоваться настройка кастомной сборки или использование через UART-мост).
*   **Внимание:** Большинство готовых Hats от Waveshare используют протокол LoRaWAN/Raw LoRa. Для работы с Meshtastic на них нужно убедиться в совместимости прошивки.

### 2.2 RAK Wireless RAK2245 / RAK5146 (Концентраторы)
*   **Описание:** Это мощные шлюзы (Gateways).
*   **Использование:** Для создания мощной базовой станции, способной слушать десятки каналов одновременно. Не подходит для бортового модуля дрона.

---

## 3. Рекомендации для Drone Swarms (Роя Дронов)

Для минимизации веса и максимизации дальности рекомендуется:

1.  **Выбор модуля:** **RAK4631** (WisBlock Core) — весит несколько граммов, работает от 3.3V, почти не греется.
2.  **Антенна:**
    *   Для дрона: Гибкая антенна-диполь ("сосиска") или легкая печатная антенна.
    *   Для базы: Коллинеарная антенна 5.8 dBi - 8 dBi, установленная на мачте.
3.  **Питание:** Подключать напрямую к BEC (Voltage Regulator) дрона на 3.3V или 5V (в зависимости от модуля).

---

## 4. Схема подключения (Wiring)

### Вариант А: USB (Самый простой)
*   Модуль (Heltec/T-Beam) -> Кабель USB-C -> Порт USB робота.
*   Драйвер определит устройство как `/dev/ttyUSB0` или `/dev/ttyACM0`.

### Вариант Б: UART (Для прямой интеграции)
Если на контроллере нет свободных USB портов:
*   **Module TX** -> **Robot RX**
*   **Module RX** -> **Robot TX**
*   **GND** -> **GND**
*   **VCC** -> **3.3V/5V**

---

## 5. Выбор частоты (Регионы)
*   **Европа / РФ:** 868 МГц (основной), 433 МГц (второстепенный).
*   **США / Канада:** 915 МГц.
*   **Азия:** 433 МГц / 923 МГц.

*Примечание: Перед покупкой убедитесь, что все модули в вашем рое работают на одной частоте.*
